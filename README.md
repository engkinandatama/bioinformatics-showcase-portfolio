## ðŸš€ Installation and Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/engkinandatama/bioinformatics-showcase-portfolio.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd bioinformatics-showcase-portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

5. **Visit**:
   ```
   http://localhost:5173
   ```
   (or another port specified by Vite).
